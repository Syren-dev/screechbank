# ScreechBank
Just a personal project to learn the banking system

# Problem
- UI/UX
  - Hard to navigate
  - Lack consistency
- Function
  - Difficulty in performing basic functions like withdrawal,payemnt and account management
  - "Bank customers aren't stupid, your app in just confusing"
- Privacy
  - Balance is displayed while vewing transaction history
- Error Recovery
  - Help users recover and diagnose error

# Solution
- UI/ UX
  - Minimalist design
  - Use same layout, language, colour
  - Tooltips
- Function
  - Group basic functions together
  - Add search bar to look for additional functions
- Privacy
  - Toggle box to toggle the view of account balance

# MVP Functions
- Minimalist design with minimal colours, with pop-up tool tips, privacy toggle for account balance, group section for common functions, error messages and reccommendation steps with error code, account summary with expenses according to timeline, calculation for loans/FDs.

# Tech Stack
- Frontend: Jsp, React.js
- Backend: SpringBoot, Lombok
- Database: MySQL
